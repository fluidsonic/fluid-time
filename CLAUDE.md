# fluid-time

Kotlin multiplatform date & time library extending kotlinx-datetime.

## Build

```shell
./gradlew build
```

## Conventions

- Tab indentation in Kotlin and Gradle files
- Tags have no `v` prefix (e.g. `0.18.0`)
- Source layout: `sources/` (common), `sources-jvm/` (JVM), `tests/` (common tests), `tests-jvm/` (JVM tests), `tests-js/` (JS tests)
- Package: `io.fluidsonic.time`
